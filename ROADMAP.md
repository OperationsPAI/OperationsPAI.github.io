# OperationsPAI Roadmap

This roadmap outlines our vision for building a self-evolving RCA training ecosystem.

## Vision

Create the world's first **self-evolving end-to-end model training environment** for Root Cause Analysis in microservices, where intelligent fault injection continuously generates challenging scenarios to improve RCA algorithms.

## Timeline Overview

```
2026 Q1-Q2: Foundation & Community Launch
2026 Q3-Q4: Intelligent Loop & First Users
```
