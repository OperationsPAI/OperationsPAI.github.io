## Support

### Getting Help

**For usage questions**:
1. Check the [documentation](https://operationspai.github.io)
2. Search [GitHub Discussions](https://github.com/OperationsPAI/operationspai.github.io/discussions)
3. Ask a new question in GitHub Discussions

**For bug reports**:
1. Check [existing issues](https://github.com/OperationsPAI)
2. Create a new issue with our [bug report template](../../.github/ISSUE_TEMPLATE/bug_report.md)

**For feature requests**:
1. Discuss in [GitHub Discussions](https://github.com/OperationsPAI/operationspai.github.io/discussions) first
2. Create an issue with our [feature request template](.github/ISSUE_TEMPLATE/feature_request.md)

### Contributing

**Want to contribute?**
1. Read the [Contributing Guide](../../CONTRIBUTING.md)
2. Find a [Good First Issue](https://github.com/search?q=org%3AOperationsPAI+label%3Agood-first-issue+state%3Aopen&type=issues)
3. Ask questions in [GitHub Discussions](https://github.com/OperationsPAI/operationspai.github.io/discussions)
